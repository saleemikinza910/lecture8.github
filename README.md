 ✨ NOVA GREET · Interactive Profile + Game App

A modern interactive web application featuring a **dynamic greeting system, profile stats, theme switcher, and mini games (including an adult-exclusive game).

Built using HTML, CSS, and Vanilla JavaScript.

---

 📌 Project Overview

NOVA GREET is a stylish and interactive web app where users can:

- Enter their name and age
- Get a personalized greeting and profile
- View dynamic stats (age status, birth year, zodiac vibe)
- Play mini games (reaction test + adult-only number guess game)

This project demonstrates advanced UI design and JavaScript interactivity.

---

✨ Features

👤 1. Dynamic Profile System
- Real-time greeting based on name
- Avatar generated from first letter
- Lucky color based on name
- Motto message based on name length

---

🎂 2. Age & Stats System
- Detects:
  - Adult / Minor status
- Displays:
  - Birth year
  - Zodiac vibe (fun logic)
  - Life progress ring (%)

---

🌙 3. Theme Toggle
- Dark / Light mode switch
- Smooth UI transitions

---

🎮 4. Reaction Test Game
- Click when button turns green
- Measures reaction time in milliseconds
- Stores best score using LocalStorage

---

🔞 5. Adult Exclusive Game (18+ Only)
- Guess the number (1–20)
- Shows only if age ≥ 18
- Tracks attempts
- Provides hints:
  - Too high / Too low
  - Correct guess

---

⚙️ 6. Action Buttons
- Clear inputs
- Example autofill
- Surprise random generator
- Copy profile info to clipboard
